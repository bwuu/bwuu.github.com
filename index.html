Fluid Simulator

This is a fluid simulator that I'm also writing in C++. It uses the smoothed particle hydrodynamics (SPH) method. Currently, it implements Weakly Compressible SPH and ghost particles at boundaries, as far as simulation quality goes. It uses marching cubes and OpenGL to produce visual output. This project was started a while ago (when I was just starting to program), so the source code is somewhat messy, as I haven't gotten around to fixing it yet.
