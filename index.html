Fluid Simulator

This is a fluid simulator that I'm also writing in C++. It uses the smoothed particle hydrodynamics (SPH) method. Currently, it implements Weakly Compressible SPH and ghost particles at boundaries. For rendering, I've already implemented a marching cubes algorithm with OpenGL, but it's quite CPU intensive so I had to give a lot of render quality with it. I'm currently writing a new renderer based on this paper: http://www.cs.rug.nl/~roe/publications/fluidcurvature.pdf. It's based on using point sprites to splat the particles and then screen space filters to make it look realistic. I've implemented the point sprites and depth extraction so far:

Next is to implement the filters to make it smooth, translucent etc.

For reference, here are some images with the old marching cubes version:

This was probably the first programming project I ever undertook, so the code is realllllly messy in some places. I remember trying to optimize with all kinds with pre-computation and whatnot, before I learned that the compiler actually does it all for you. I'll probably go back and refactor the code at some point, after I get better rendering working.
